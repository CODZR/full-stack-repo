# 快捷键

## vscode

### 常用
```
Command + 点击 多光标插入
Shift + 点击 复制选中的代码块
Command + X 剪切（未选中文本的情况下，剪切光标所在行）
Command + C 复制（未选中文本的情况下，复制光标所在行）
Command + V 粘贴
Command + L 选中当前行
Command + A 全选

Command + Z 返回上一步
Command + Shift + Z 撤销返回

Option + Up 向上移动行
Option + Down 向下移动行
Option + Shift + Up 向上复制行
Option + Shift + Down 向下复制行

Command + Enter 下一行插入
Command + Shift + Enter 上一行插入
Command + Shift + \ 跳转到匹配的括号

Command + Up 跳转至文件开头
Command + Down 跳转至文件结尾
Command + Shift + Up 选中光标所在位置至文件开头
Command + Shift + Down 选中光标所在位置至文件结尾

Control + G 跳转至某行

command + option + left tab左切换
command + option + right tab右切换

fn +Up 按屏向上滚动
fn +Down 按屏向下滚动

```

### 高级
```
Command + Shift + P 显示命令面板
Command + P 快速打开某文件
Command + Shift + C 打开新终端
Command + Shift + N 打开新的vscode窗口（在当前项目的基础上）

Command + N 新建文件
Command + O 打开文件
Command + S 保存文件
Command + Shift + S 另存为
Command + Option + S 全部保存
Command + W 关闭当前文件
Command + \ 编辑器分屏
Command + / 添加注释

Command + F 查找
Command + Option + F 替换

Command + K， Command + W 关闭全部已打开文件
Command + K， Enter 保持文件打开
Command + K， P 复制当前文件路径
Command + K， R 在资源管理器中查看当前文件
Command + K，O 新窗口打开当前文件
Command + K， V 分屏显示 Markdown预览窗口
command + k，left（right）分屏光标左右切换。如果关闭分屏  ，光标跳转到要关闭的分屏，执行command + w

```
